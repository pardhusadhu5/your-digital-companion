# 🚀 AI Tools Discovery Platform

A modern web application that helps users **discover, explore, and compare AI-powered tools** across multiple categories.

🌐 Live Demo: http://inspiring-liger-a61c26.netlify.app/

---

## 📌 Overview

This project is a **curated AI tools directory platform** where users can:

- 🔍 Search for AI tools
- 📂 Browse tools by category
- ⭐ Discover trending or featured tools
- ⚡ Explore productivity, design, and developer utilities

The goal is to make AI tool discovery **fast, simple, and user-friendly**.

---

## ✨ Features

- 🔎 Smart search functionality
- 🧠 AI tools categorization (Productivity, Design, Developer Tools, etc.)
- 📊 Clean UI for easy comparison
- ⚡ Fast and responsive design
- 🌐 Deployed on Netlify
